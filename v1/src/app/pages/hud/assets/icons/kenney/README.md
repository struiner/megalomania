# Kenney Game Icons Directory

Primary source for HUD icons (CC0 1.0 license).

## Usage
This directory contains the selected icons from Kenney's Game Icons pack, specifically chosen for HUD functionality and recolored to match the Hanseatic theme (brass stroke on transparent background).

## Selected Icons for HUD Pilot
- backpack.png (Inventory)
- ledger.png / book.png (Ledger)
- map.png (Maps)
- group.png (Crew)
- scales.png (Trade)
- compass.png (Quests) / star.png (fallback)
- cog.png (Settings)
- question.png (Help)
- anchor.png (Status header)
- bell.png (Notifications header)

## Processing Notes
- Source: Kenney.nl Game Icons (CC0)
- Recolor: Brass stroke (#c08a3b) on transparent background
- Sizes: 16px native, 32px 2x scale
- Format: PNG with hard edges, no anti-aliasing
